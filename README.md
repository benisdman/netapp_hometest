## Setup Backend:

0. `cd BE`
1. `virtualenv netapp_hometest_env`
2. `source netapp_hometest_env/bin/activate`
3. `pip install -r requirements.txt`
4. `python app.py`


## Setup Frontend
0. `cd FE`
1. `npm install`
2. `npm run start`

App on `localhost:3000`
